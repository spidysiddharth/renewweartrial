Renew Wear

Project Analysis Dashboard

Executive Summary

Smarter, Sustainable Student Living

Renew Wear is a purpose-driven online marketplace designed specifically for the college ecosystem. It directly addresses the intersection of sustainability, affordability, and the unique needs of students. By integrating advanced technologies like AI for dynamic pricing and secure verification systems, we offer a seamless, tech-driven solution that transforms how students buy and sell second-hand goods. Our goal is to reduce waste, foster community trust, and establish a thriving circular economy on every campus.

🎯 Our Motto

"Sustaining Student Life – One Reused Item at a Time."

⚠️
🚨

The Campus Waste Crisis

  • ✖️ High Student Turnover: Graduations and internships lead to frequent relocations and massive item abandonment at the end of each semester.
  • ✖️ Informal & Unsafe Trading: Reliance on unverified WhatsApp groups or word-of-mouth is unreliable, inconvenient, and lacks structural trust.
  • ✖️ Environmental Degradation: Perfectly usable goods—like bicycles, electronics, and dorm furniture—end up in landfills, creating unnecessary carbon emissions.
🌱
💡

The Renew Wear Solution

  • ✔️ Centralized Resale: A dedicated C2C and C2B2C marketplace tailored exclusively for the student ecosystem, keeping value within the campus.
  • ✔️ Smart & Secure: AI-powered valuation for fair pricing, verified user profiles via Clerk Auth, and robust data security infrastructure.
  • ✔️ Eco-Conscious Logistics: Hyper-local peer-to-peer exchanges and planned 24/7 smart lockers drastically minimize transportation emissions.

System Architecture

Platform & Technology

This section details the robust technological foundation of the Renew Wear SPA. It organizes the platform's features by mapping the tech stack to specific user benefits. Users can interact with the architecture components to understand how AI, cloud databases, and modern auth frameworks create a secure and seamless experience.

🔐
→

Clerk Authentication

Identity verification using OTPs and Google OAuth.

Provides seamless login via OTP and OAuth. Enforces Role-Based Access Control (RBAC) separating Buyers, Sellers, and Admins. Data secured with AES-256 encryption.
🗄️
→

MongoDB

Flexible NoSQL database managing users, products, addresses, and orders.

Utilized for its schema-less flexibility. Core collections include: Users, Products, Addresses, and Orders. Allows real-time tracking from the Seller Dashboard.
🧠
→

Deep Learning (ResNet)

Visual recommendation system for finding similar fashion items.

Users upload an image. OpenCV pre-processes it, and a pre-trained ResNet CNN extracts feature vectors. Cosine similarity calculates matches with >90% accuracy in <2 seconds.
📸
→

Cloudinary CDN

Optimized image hosting and delivery API.

Offloads image storage from the main server. Provides automatic format optimization and rapid global delivery via CDN, crucial for a visual e-commerce interface.
⚡
→

Node.js, React.js & Vercel

Server-side rendering, CI/CD, and serverless functions.

Node.js ensures SEO and fast load times. Vercel provides seamless deployment, handling serverless backend APIs for order processing without dedicated server management.
📧
→

SMTP Auto Mail

Automated transaction and verification workflows.

Triggers instant email notifications for order placements, delivery status updates, and seller alerts, ensuring transparent communication without manual intervention.
